Privacy Policy

MidnightMics Content Tool

1. Overview

MidnightMics Content Tool is a personal content management application used by the creator behind @midnightmics on TikTok. This privacy policy explains what data the tool handles and how it is stored.

This is a personal tool operated by a single individual. It is not a public service, and it does not have external users.

2. Data We Collect

This tool collects and stores the minimum data necessary to function:

  • TikTok OAuth tokens — Used to authenticate with TikTok's API and publish content to the @midnightmics account. These tokens are obtained through TikTok's official OAuth authorization flow.
  • Basic account information — Limited profile information (such as username and account ID) returned by TikTok during the OAuth process, used solely to confirm the correct account is connected.

That is the full extent of data collection. The tool does not collect analytics, usage data, cookies, IP addresses, or any other personal information.

3. How Data Is Used

All collected data is used exclusively for one purpose: publishing video content to the connected TikTok account via the TikTok Developer API. Data is never used for advertising, profiling, analytics, or any other secondary purpose.

4. Data Storage and Security

OAuth tokens and account information are stored on a private, access-controlled server operated by the tool's owner. Specifically:

  • Tokens are stored in encrypted or access-restricted configuration files
  • The server is not publicly accessible
  • Access is limited to the tool's owner
  • No cloud databases or third-party storage services are used for sensitive credentials

5. Third-Party Sharing

Data is never shared with, sold to, or disclosed to any third party. The only external service the tool communicates with is the TikTok API, as required to publish content.

6. Data Retention

OAuth tokens are retained only as long as they are needed to operate the tool. Tokens can be revoked at any time through TikTok's account settings, which immediately removes this tool's access.

7. Children's Privacy

This tool is not a public-facing service and does not collect data from any users, including children. It has no user registration, no public interface, and no mechanism for anyone other than its owner to interact with it.

8. Your Rights

Since this tool is operated by and for a single individual (the account owner), traditional data subject rights (access, deletion, portability) are exercised directly by the owner through server access and TikTok account settings.

9. Changes to This Policy

This policy may be updated if the tool's functionality changes. Updates will be reflected on this page with a revised date.

10. Contact

For questions about this privacy policy or the tool's data practices, contact: midnightmics [at] proton [dot] me
